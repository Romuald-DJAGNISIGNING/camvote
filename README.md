# CamVote – Secure Digital Electoral System

CamVote is a cross-platform (Android, iOS, Web) civic-tech application designed to promote transparent, secure, and fraud-resistant elections in Cameroon. It is an e-Voting platform designed to modernize and protect Cameroon’s electoral process. It allows citizens to register using biometrics, vote remotely, and view real-time results — all with full transparency and legal compliance.
